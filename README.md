# SemSync

SemSync is a comprehensive academic organization platform designed to streamline scheduling, communication, and resource sharing for university students and instructors.

## Project Structure

This repository is organized into the following main components:

### Web Platform (`/web-platform`)

The core web-based application built with React and Firebase. It currently serves as the primary interface for the platform, offering features such as:

- Interactive Dashboards for students and instructors.
- Real-time Timetable synchronization.
- Academic Group management and resource sharing.
- Task tracking and digital notebooks.

Please refer to the [Web Platform Documentation](web-platform/README.md) for detailed installation instructions and technical documentation specific to the web application.

### Mobile Application (Coming Soon)

Development for the dedicated mobile application (iOS/Android) is scheduled for the next phase of the project. This directory will contain the source code for the mobile client, likely utilizing cross-platform technologies to ensure broad compatibility.
